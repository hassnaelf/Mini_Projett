****************************************************Site Web pour la gestion de ESTE ********************************************************************************
Notre site web concerne trois gestion principaux :

1 .  La gestion des filières
2 .  La gestion des étudiants
3 .  La gestion des Professeurs

Pour  Réaliser ce site web  On a travaillé  avec   :
 
        -  La base données   Mongodb ATLAS
		-  Node
	    -  React js
		-  Axios (Pour faire des requete http)
		
La struture de notre site web est comme suit :

Pour accèder au service offertes par notre site web l'administrateur doit tout d'abort s'identifier en saissisant  le nom d'utilisateur et le mot de passe 
    Remarque  : (Username  :  admin && mot de passe : 123)
	
Après avoir s'identifier l'administrateur a la possibilité de faire un certain nombre de gestion à titre d'exemple , il peut faire  différents traitement sur les filières ;
il a la possibilité d'ajouter une formation (ISIL ,DUT ,Master ....)
la liste des formation va etre configurer automatiquement dans le formulaire d'ajout des filières ;cette dernière opération nécessite que l'administrateur remplit les informations 
realatives au filière à ajouté (formation ,nom de filière, Description ,capacité ...) ensuite il valide l'insertion , après cette opération il se dirige automatiquemet vers une page 
dont lequelle les informations relatives aux filières sont affichés .Cette page offre la possibilité à l'administrateur de faire d'autres traitement comme la modification et la 
suppression des filières .

le site web donne le droit à l'administrateur de faire la gestion des professeurs et leurs affectés les matières à enseigner .
Par ailleurs , l' administrateur peut gérer les étudiants .
